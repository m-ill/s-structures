S-Structures 사용 방법

3D 화면 조작
· 우클릭 드래그 = 회전(궤도), Shift+우클릭 = 화면 이동
· 휠 = 확대/축소, 선택 모드에서 빈 곳 드래그 = 회전
· 상단 [평면/정면/측면] 버튼으로 빠른 시점 전환

구조 모드 — 작업평면(시점에 따라 자동 선택) 위에 모델링:
· 부재: 드래그로 작성 — 시작점에서 X/Y/Z 축 방향 자동 스냅
  (가이드선 표시, 격자 길이 단위) + 격자·절점 스냅
· 다중 선택: 드래그 박스로 절점·부재 한번에 선택 (Shift=추가)
  → 단면·재료·접합·지점 일괄 변경, 동일 하중(분포 등) 일괄 입력,
  하중 일괄 삭제, 요소 일괄 삭제 (Delete키)
· 절점 추가: 부재 위 클릭 → 그 자리에 절점 생성·부재 분할
  → 그 절점의 변위 확인, 절반 삭제(잘라내기), 거기서 이어 그리기
· 부재 위 스냅: 기존 부재 중간점에 스냅(보라 링) →
  그 자리에서 그리면 부재 자동 분할·연결 (하중 재배분)
· 작업면 패널: XY/XZ/YZ 선택, 오프셋 ±, 선택 절점에 맞춤
  → 어느 층·어느 입면에서든 그리기 가능
· 기둥: 절점/격자점 클릭 → 층고만큼 수직 부재 생성
· ⬆ 층 복사: 최상층(기둥·보·하중)을 한 층 위로 복제 —
  여러 번 누르면 2층, 3층… (보 평면만 있으면 기둥 자동 생성)
· 지점: 절점 클릭 — 회전단(핀)·이동단(롤러)·고정단
  절점 속성에서 사용자 정의(6자유도): Dx·Dy·Dz·Rx·Ry·Rz
  개별 구속/해제 (모멘트 잠금·풀기)
· 부재 단부 접합: 부재 속성에서 단부₁/단부₂를
  강접합(라멘) / 핀접합(모멘트 해제, ○표시) 선택
· 하중: 절점·부재 클릭, 방향은 속성 패널(±X/±Y/±Z)
· 하중 스텝 재생: 하중마다 스텝 번호 지정 → 우하단 [▶ 스텝 재생]
  스텝이 누적되며 0→100% 증가하는 과정이 애니메이션으로 표시
  (변형·단면력·검정 색 실시간 변화, 슬라이더로 스크럽)
· 절점 드래그 = 수평 이동, Shift+드래그 = 상하(Z) 이동

단면검정 (허용응력) — [검정] 토글:
· 검정비 = |N|/(A·fb) + |Mz|/(Zz·fb) + |My|/(Zy·fb), 전단 별도
· 초록 OK(≤0.7) / 노랑 주의(≤1.0) / 빨강 NG(>1.0)
· 좌굴·비틀림 검토는 포함되지 않음 (참고용)

간단 탄성설계 모드 — [설계]/[처짐] 토글 + 메뉴의
하중케이스·조합 설정 / 설계 요약 보고서:
· 하중케이스(D/L/풍/지진)와 조합 계수를 직접 편집
· 부재 속성의 설계 섹션에서 구분(보/기둥/가새)·Ky·Kz·Lb·처짐한계 설정
· OK=초록(검정비<0.7), WARN=주황(경고/0.7~1.0), NG=빨강(>1.0 또는 세장비·처짐 초과)
· 자중은 설정에서 켜면 D 케이스에 자동 포함 (model에 저장되지 않음)
이 기능은 교육·초기 검토용 간단 탄성설계 검토입니다. 좌굴, 횡좌굴, 접합부, 상세 기준 검토는 사용자가 별도 확인해야 하며 최종 구조설계 계산서를 대체하지 않습니다.

해석: 변경 시 자동 실행. 변형도/M/Q/N/반력 토글 표시.
M은 강축 Mz(빨강)+약축 My(보라), T=비틀림. 절점#/부재#/로컬축 토글.
하중조합 선택(기본 CO1/SLS1) + 포락(Envelope) 표시.
JSON v3: 단위·재료·단면·로컬축·조합·해석설정·결과 포함 (v1/v2 자동 변환)

그리기 모드: 작업면(그리드) 위에 그려져 모델과 함께 회전·확대
· 자유=손글씨, 직선(그리드)=격자·절점 스냅 직선 (길이 표시)
· 펜 색상/굵기는 상단 바에서 선택
(Apple Pencil 등 스타일러스 지원)

태블릿: 한 손가락=현재 도구, 두 손가락=화면 이동+핀치 줌,
우하단 ⟳/✥ 버튼으로 한 손가락 드래그를 회전↔이동 전환, ⛶=모델 맞춤
선택 모드에서 빈 곳 드래그=회전. 브라우저에서 파일을 열거나
"홈 화면에 추가"로 앱처럼 사용
