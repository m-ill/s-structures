2. **태블릿(iPad 등)**: 사파리/크롬에서 열고 **공유 → 홈 화면에 추가** → 앱처럼 전체화면 실행.

| 동작 | PC | 태블릿 |

| 화면 회전 | 우클릭 드래그 / 빈 곳 드래그(선택 모드) | 한 손가락 드래그(선택 모드) |

## 📐 해석 사양

- **요소**: 3D 프레임 (축력·비틀림·2축 휨, 12×12 강성행렬, 정적 응축에 의한 단부 해제)
- **결과**: 부재당 21+ 내부 스테이션, 부재하중 고정단 변형 기여 포함한 정확한 처짐곡선
- **검정식**: `|N|/(A·Fa) + |Mz|/(Zz·Fb) + |My|/(Zy·Fb) ≤ 1` (전단 별도) — 허용응력도법
- **검증**: 단순보·고정보·캔틸레버·포털·3D 프레임 이론해 벤치마크 및 3층 골조
  회귀 테스트(총하중=총반력, 평형 잔차 < 1e-6) 100+건 통과
- **재료/단면**: SS400·SM490·콘크리트·목재 / H형강·각형강관·강관·RC·목재 단면 내장

## 📐 간단 탄성설계 모드 사용법

1. 상단 **[설계]** 토글을 켜면 부재가 OK(초록)/WARN(주황)/NG(빨강)로 표시됩니다. **[처짐]** 토글은 보의 L/Δ를 표시합니다.
2. 메뉴(☰) → **하중케이스·조합 설정**: D(고정)/L(활)이 기본이며, 풍하중 4방향·지진 EX/EY를 버튼으로 추가하고 조합 계수를 직접 편집하거나 예비 프리셋을 생성합니다.
3. 부재를 선택하면 속성 패널의 **설계** 섹션에서 구분(보/기둥/가새)·**Ky·Kz**(유효좌굴길이계수)·**Lb**(비지지길이)·처짐한계(L/x)를 설정합니다. 다중 선택으로 일괄 입력도 됩니다.
4. **설정** → "자중 자동 포함"을 켜면 부재 자중이 D 케이스에 자동 반영됩니다.
5. 메뉴(☰) → **설계 요약 보고서**: OK/WARN/NG 통계, 부재별 검정비·지배조합·KL/r·L/Δ 표를 확인하고 인쇄(PDF)할 수 있습니다.

**판정 의미**: OK = 검정비 0.7 미만 · WARN = 0.7~1.0 또는 주의사항(Lb 미설정, 비틀림 존재 등) · NG = 검정비 1.0 초과 또는 세장비·처짐 한계 초과.

## ⚠️ 사용 범위

본 도구는 **간이 검토·교육·아이디어 스케치**용입니다. 선형 정적해석이며, 간단 탄성설계 모드의
검토(응력·KL/r 스크리닝·처짐)는 **교육·초기 검토용**입니다. 좌굴 내력 저감, 횡좌굴(LTB), 접합부,
국부좌굴, 2차효과(P-Δ)는 설계되지 않으며, 하중조합 프리셋은 설계기준의 공식 조합이 아닙니다.
**최종 구조설계 계산서를 대체하지 않습니다** — 실시설계는 반드시 정식 절차로 확인하세요.
