## ✨ 무엇을 할 수 있나요

**노트에 그림 그리듯 3D 골조를 그리면, 그리는 즉시 해석·검정됩니다.**

| | 기능 |
|---|---|
| 🖊️ | **스케치 모델링** — 격자·절점·부재 위 스냅, X/Y/Z 축 자동 스냅(CAD ortho), 작업면 선택(XY/XZ/YZ+오프셋) |
| 🏢 | **층 복사** — 1층을 그리고 버튼 한 번에 2층, 3층… 보 평면만 그리면 기둥 자동 생성 |
| ⚡ | **실시간 3D 해석** — 직접강성법, 절점당 6자유도, 변경 즉시 재해석 |
| 🔗 | **실무형 경계조건** — 회전단·이동단·고정단 + 6자유도 개별 구속, 부재 단부 강접합/핀접합(트러스 가능) |
| ✅ | **단면검정** — 허용응력 검정비 색상 표시(초록/노랑/빨강 NG), 부재별 상세 내역 |
| 📊 | **결과 표시** — 변형도(부재 내부 처짐 포함), Mz/My·전단·축력·비틀림 다이어그램, 반력, 포락(Envelope) |
| 🧮 | **하중조합** — 케이스별/조합별 해석, 강도·사용성 조합, 평형 잔차 자동 확인 |
| ✍️ | **손글씨 메모** — 작업면 위에 펜 메모(자유곡선/그리드 직선), 모델과 함께 회전·확대 |
| 📄 | **PDF 내보내기** — 현재/전체 페이지를 PDF 파일로 바로 다운로드 |
| 💾 | **JSON 저장** — 단위·재료·단면·하중조합·해석설정·결과까지 파일 하나에 (스키마 v3) |

## 🚀 시작하기

1. **PC**: 위 링크를 크롬/엣지/사파리에서 열기 — 끝.
2. **태블릿(iPad 등)**: 사파리/크롬에서 열고 **공유 → 홈 화면에 추가** → 앱처럼 전체화면 실행.

### 기본 조작

| 동작 | PC | 태블릿 |
|---|---|---|
| 화면 회전 | 우클릭 드래그 / 빈 곳 드래그(선택 모드) | 한 손가락 드래그(선택 모드) |
| 이동 / 확대 | Shift+우클릭 / 휠 | 두 손가락 드래그 / 핀치 |
| 부재 그리기 | 구조 모드에서 드래그 (축 가이드 자동) | 동일 |
| 실행취소 | Ctrl+Z / Ctrl+Y | 상단 ↶ ↷ |

열면 3D 포털 프레임 샘플이 해석된 상태로 시작합니다. 도움말(☰ → 도움말)에 전체 사용법이 있습니다.

## 📐 해석 사양

- **요소**: 3D 프레임 (축력·비틀림·2축 휨, 12×12 강성행렬, 정적 응축에 의한 단부 해제)
- **결과**: 부재당 21+ 내부 스테이션, 부재하중 고정단 변형 기여 포함한 정확한 처짐곡선
- **검정식**: `|N|/(A·Fa) + |Mz|/(Zz·Fb) + |My|/(Zy·Fb) ≤ 1` (전단 별도) — 허용응력도법
- **검증**: 단순보·고정보·캔틸레버·포털·3D 프레임 이론해 벤치마크 및 3층 골조
  회귀 테스트(총하중=총반력, 평형 잔차 < 1e-6) 100+건 통과
- **재료/단면**: SS400·SM490·콘크리트·목재 / H형강·각형강관·강관·RC·목재 단면 내장
